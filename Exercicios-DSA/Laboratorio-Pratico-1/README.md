Dashboard desenvolvido para analisar os dados de vendas de um determinado período em uma empresa fictícia, utilizando ferramentas de ETL e DAX (Power BI) para visualização de indicadores-chave (KPIs).

Itens analisados no Dasboard.
1. Soma do valor total vendido
2. Quantidade de vendas por categoria de produto.
3. Quantidade de vendas por país considerando a prioridade de entrega.
4. Média de descontos nas vendas por subcategoria dos produtos
5. Calculo da média do valor das vendas por pais.

Umas das funcionalidades do dashboard é a possibilidade do usuário filtrar os dados apresentados por ano, por segmento e por país de acordo com suas necessidades.
